The Game of Life (1P Mode)

  • High Concept: A variation of Conway's Game of Life in which the player can interact with the game state.
  • Player Experience Goals: Captivation.
  • Genre: Puzzle.
  • Game play: Player can click on any "life tile" on the grid. A light tile is alive and a dark tile is not alive. Clicking on an alive tile kills it and clicking on a tile thatis not alive gives it life. If there is only 1 or 0 alive tiles left the game is lost.
  • Features: It is a spin on a popular game, in which a player can now interact with the interesting aspects of the original Game of Life.
  • Prototype Closest Other Game: Conway's Game of Life.
